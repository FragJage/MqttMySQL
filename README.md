# MqttMySQL
Mqtt daemon to log mqtt message into MySQL.

## Introduction
This daemon log mqtt message or value into MySQL database.

## Features
 - Settings MySQL
 - Settings each topic
 - TO DO support json message

## Building
git submodule update --init --recursive
cmake .
make
